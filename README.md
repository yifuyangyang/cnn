（4）实验说明
本实验对比了多层感知机（MLP）和卷积神经网络（CNN）在MNIST手写数字数据集上的分类性能。MLP采用一个包含两层的全连接网络结构（输入层784→隐藏层256→输出层10），通过Flatten操作丢失空间信息，最终测试准确率为98.03%。而CNN采用两个卷积层（卷积通道数为16→32，每层后接ReLU和2×2最大池化）加全连接层的结构，有效利用图像局部特征和空间结构，最终测试准确率达到99.09%。实验表明，在图像分类任务中，CNN因其局部感知、权值共享和池化特性，相比MLP具有更强的特征提取能力和更高的分类精度。
