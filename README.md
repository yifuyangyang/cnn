本实验对比了传统的BP神经网络（多层感知机MLP）和卷积神经网络（CNN）在手写数字识别任务（MNIST数据集）上的性能表现。通过调参优化，分别达到了MLP达98.03%和CNN达99.34%的测试准确率。
输入层: 784神经元（28×28图像展平）
隐藏层1: 512神经元 + ReLU激活 + Dropout(0.2)
隐藏层2: 256神经元 + ReLU激活 + Dropout(0.2)  
隐藏层3: 128神经元 + ReLU激活 + Dropout(0.2)
输出层: 10神经元（softmax分类）
卷积层1: 1→32通道，3×3卷积，ReLU，2×2最大池化
卷积层2: 32→64通道，3×3卷积，ReLU，2×2最大池化  
卷积层3: 64→128通道，3×3卷积，ReLU，2×2最大池化
展平层: 128×3×3 = 1152维
全连接层1: 1152→256，ReLU，Dropout(0.3)
全连接层2: 256→128，ReLU
输出层: 128→10（softmax分类）
